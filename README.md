# 自动领支付宝红包

这是一个手机浏览器跳转到支付宝 APP 自动领取红包的静态页面。支持 Android、iOS 自动跳转，也支持电脑端展示搜索码与二维码，省去保存二维码、扫码、复制口令再粘贴的麻烦。

## 优势

- **自动跳转**：手机端非微信浏览器打开后，2 秒倒计时自动跳转支付宝 APP 领取红包。
- **设备适配**：根据 Android / iOS 自动展示对应的引导背景图（`android.gif`、`ios.gif`）。
- **微信兼容**：微信内无法直接跳转时，点击页面即可复制红包搜索码，引导用户去支付宝粘贴领取。
- **桌面端友好**：电脑端不尝试跳转，而是显示搜索码、复制按钮和二维码，扫码即可在手机端继续领取。
- **纯静态部署**：单 HTML 文件，无需后端，可直接部署到 Cloudflare Pages、GitHub Pages 等静态托管服务。
- **配置简单**：只需修改文件顶部的三个参数即可使用。

## 使用说明

- 手机浏览器（非微信）打开页面，等待 3 秒后自动跳转。
- 微信内打开页面，点击屏幕复制搜索码。
- 电脑端打开页面，可复制搜索码或扫描二维码后在手机上领取。

## 如何获取三个参数

### 1. 支付宝商家 ID（shareId）

使用支付宝扫描下图二维码，扫码后即可获得 `shareId`。

![支付宝商家 ID 二维码](shareId.png)

### 2. 红包码 Token（toKen）

1. 打开支付宝，生成你自己的赏金二维码。
2. 用微信扫描这个赏金二维码。
3. 微信扫码后会得到一个链接，取出链接后面那一段字符串。

格式通常为：`英文字母 + 19 + 随机字符串`

例如扫码后得到：`QqCg19616423r8lcpdrtczety25Mzq`

只需要提取 `19` 及其后面的部分：

```js
const toKen = '19616423r8lcpdrtczety25Mzq';
```

### 3. 红包码搜索码（hongbaoNum）

1. 在支付宝里找到赏金红包的「分享」功能。
2. 复制分享口令。
3. 从口令中提取搜索码。

例如分享口令为：

```
H:/hEqF6H080v4 或复`zhi`此消息打开 zh`i 怤保，鸿葆藏真情，生活天天向上  y:/r MF0689 $794
```

其中 `hEqF6H080v4` 就是搜索码：

```js
const hongbaoNum = 'hEqF6H080v4';
```

## 参数配置

打开 `index.html`，修改以下三个值：

**支付宝商家 ID**

```js
const shareId = '2088222971225802';
```

**红包码 Token**

```js
const toKen = '19616423r8lcpdrtczety25Mzq';
```

**红包码搜索码**

```js
const hongbaoNum = 'hEqF6H080v4';
```
